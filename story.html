Web Assignment Summary

This web assignment demonstrates the application of HTML and CSS to build a semantically structured webpage. The focus is on proper partitioning of the document using semantic tags, designing with CSS, and embedding external content using <iframe>.

Semantic Partitioning

The webpage was organized using semantic elements to ensure clarity and accessibility:

  • <header>: Contains the site’s title and navigation bar for seamless user interaction.
  • <nav>: Includes a horizontal menu linking to different sections of the site.
  • <main>: Hosts the primary content, ensuring focus on the most relevant information.
  • <footer>: Provides copyright information and links to external resources.

CSS Styling

CSS was used to enhance the appearance of the webpage:

  • Applied a clean and modern layout with consistent spacing and font choices.
  • Styled the navigation bar with hover effects for better user experience.
  • Ensured responsiveness using flexible units and media queries.

Usage of <iframe>

The <iframe> tag was used to embed external content dynamically within the webpage. Key highlights include:

  • Embedded a YouTube video to enrich the content visually.
  • Defined specific width and height attributes for better control over the embedded content’s appearance.
  • Applied CSS styling to align the <iframe> within the layout seamlessly.

An example of the <iframe> usage:

<iframe src="https://www.youtube.com/embed/example_video_id" 
        width="560" height="315" frameborder="0" allowfullscreen></iframe>

Conclusion

This assignment highlights the importance of semantic HTML for accessibility, CSS for aesthetics, and <iframe> for dynamic content integration. The result is a structured, visually appealing, and interactive webpage.
