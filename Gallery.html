How to Insert Images in HTML

Basic Syntax

To insert an image in HTML, use the <img> tag. This tag is self-closing and requires at least the src attribute to specify the image's location.

<img src="image.jpg" alt="Description of image">

The alt attribute provides alternative text for the image if it cannot be displayed.

Example

Here's an example of embedding an image:

<img src="house.jpg" alt="House Image">

Organizing Your Files

For better organization, store your images in a separate folder, typically named images. Update the src path accordingly.

<img src="images/photo.jpg" alt="A beautiful photo">

Adding CSS for Styling

You can style images using CSS. For instance:

img {
    max-width: 100%;
    height: auto;
    border: 2px solid #ddd;
    border-radius: 5px;
}

This ensures the image is responsive and styled nicely.
